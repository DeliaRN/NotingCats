#Noting Cats

Es una app de notas sencillita (porque en cuatro días no puedo hacer más, que hay que estudiar para los exámenes.)

Permite crear notas, editarlas y borrarlas.
Quedó a medio implementar un sistema de iconos para las notas a modo de identificador, y un selector de colores para las mismas.


## Librerías y herramientas usadas:

-Android y Kotlin para desarrollar la aplicación. Extensiones core.ktx compatibilidad con appcompat.
-Para la arquitectura, MVVM, con ViewModel, LiveData y Room
-Para la interfaz de usuario y animaciones, Material Design, Constraint Layout, SpashScreen y Lottie para el sistema de animación y partículas.
-Para la navegación, Android Navigation Component y ViewBinding.
-Para el almacenamiento local de datos, Room, con una capa de abstracción sobre SQLite, que accede a la bd mediante entidades, DAO y consultas reactivas.
-Compilado con Gradle


## Cómo ejecutar el proyecto:

1. Clona el repositorio o descomprime el archivo ZIP.
2. Abre la carpeta del proyecto en Android Studio.
3. Espera a que se sincronicen todas las dependencias.
4. Ejecuta la app en un emulador o dispositivo Android (Shift + F10).
Requiere Android Studio Hedgehog o superior.
